# 🔐 SecureVault — Your Digital Password Guardian

⚡ **Store. Encrypt. Protect.**

A secure password management application designed to safely store credentials, generate strong passwords, and protect sensitive information using encryption techniques.

No more remembering multiple passwords.
No more insecure storage.

🛡️ **From simple passwords → secure digital vault**

---

# 🧠 The Idea

Today, users manage multiple accounts:

📧 Emails
💳 Banking accounts
🌐 Social media
💻 Applications

The biggest problems:

❌ Weak passwords
❌ Reusing passwords
❌ Storing passwords in unsafe places

## 💡 Solution

SecureVault provides a centralized and secure platform where users can:

* Store passwords safely
* Encrypt sensitive information
* Generate strong passwords
* Manage credentials easily

# 📦 Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/SecureVault.git
```

## Navigate to Project

```bash
cd SecureVault
```

---

## Run Java Version

```bash
javac PasswordManager.java

java PasswordManager
```

---

## Run Python Version

```bash
python main.py
```

---

## Compile C

```bash
gcc password_manager.c -o vault

./vault
```

---

## Compile C++

```bash
g++ PasswordManager.cpp -o vault

./vault
```
